# Halex
A 2D top-down exploration game written in Haskell.  
*Take control of Halex to explore an underground labyrinthh, discover three haunting ruins, and escape to the surface.*  

## Controls:

**W**         - Move North  
**A**         - Move West  
**S**         - Move South  
**D**         - Move East  
**Space**     - Use Exit  
**Esc**       - Quit Game
